📋 How to Replace Any Graphic

🖼️

Option 1 — Upload Photo

Click any image card → "Replace" → upload your JPG, PNG or WebP photo. It replaces the graphic instantly on the website.

⚙️

Option 2 — Edit SVG Code

Go to SVG Editor tab → select graphic → modify colors, shapes, text → Apply. Great for tweaking existing graphics.

🎨

Option 3 — Color Customizer

Go to Color Customizer → change your brand colors → all SVG graphics update automatically to match.
